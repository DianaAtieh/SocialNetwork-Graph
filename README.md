# Social Network Graph (Java)

## 📌 Description
This project implements a simple Social Network using a Graph data structure in Java.

Users are represented as vertices, and relationships between users are represented as edges. The system supports adding users, connecting them, and handling different edge cases using custom exceptions.

---

## 🚀 Features
- Graph-based social network implementation
- Add/remove users (vertices)
- Connect users (edges)
- Custom exception handling:
  - User already exists
  - User not found
  - Vertex already exists
  - Vertex not found
- Clean OOP design

---

## 🧠 Concepts Used
- Graph Data Structure
- OOP (Classes, Objects)
- Exception Handling
- Encapsulation
- Data abstraction

---

## 📂 Project Structure
- `Graph.java` → Graph implementation
- `SocialNetwork.java` → Core logic
- `Main.java` → Runs the program
- `UserExistException.java`
- `UserNotFoundException.java`
- `VertexExistException.java`
- `VertexNotExistException.java`

---

## ▶️ How to Run
1. Open the project in IntelliJ
2. Run `Main.java`

You can simulate a social network where users connect with each other and manage relationships dynamically.
